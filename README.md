# PSUSphere

PSUSphere is a Django-based web application for managing student organizations, academic programs, and membership information. It provides a foundational structure to store colleges, programs, organizations, students, and organization membership data.

## Features

- Manage colleges and academic programs
- Create and maintain student organization profiles
- Track students and their program enrollment
- Manage student memberships within organizations
- Includes a management command to seed initial demo data
